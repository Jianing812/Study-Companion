# Study-Companion
Study Companion is an adaptive AI study tool that transforms notes, audio, and images into personalized exam preparation formats based on different learning needs.

## Problem
Most study tools assume every student learns the same way. During exam season, this creates barriers for students who:
- learn better through audio
- struggle with focus during long study sessions
- need simpler academic language without losing technical accuracy

## Solution
Study Companion allows users to upload text, audio, or images and convert them into personalized study modes such as:
- **Listen Mode** for audio-based learners
- **Focus Mode** for short, interactive study challenges
- **Clarity Mode** for simplified but accurate academic explanations

## Features
- Upload notes, slides, screenshots, or lecture audio
- Convert content into adaptive learning formats
- Generate concise study summaries
- Create quiz-based checkpoints
- Support more accessible and personalized exam prep

## Target Users
- Multi-Modal Learners
- Focus-Seeking Students
- Global Scholars

## Tech Stack
- React / Next.js
- Tailwind CSS
- OpenAI API
- Speech-to-text / text-to-speech tools

## Project Status
This project is currently being developed as a case competition prototype.
